## 目标

根据 `SprayPaintPlan.cellRegions` 与地形高度场，为每个 `regionId` 生成 **负向遮挡罩** 薄壳 mesh（`SprayMaskMeshPayload`），满足与 `Terrain_Main` 同坐标系、可水密校验；并在预览中可加载显示（静态列表，套合交互在任务-11）。

## 范围

### 包含

- 高度场薄壳算法：包络顶面 + 开窗 + 侧壁 + 底缘
- `bleedMarginMm` 格点级侵蚀/膨胀（P1）
- IPC：`spray:generate-masks`
- 罩体列表数据传入预览组件（名称、`Mask_Color_XX`、regionId）
- `assertWatertightMesh` 校验；陡崖 > 70° UI 警告数据

### 不包含

- 点击套合、视图模式切换（任务-11）
- STL 写文件与 ZIP（任务-12）
- 一体定位框/底座（P2，后续）

## 技术要点

### 1. 输入

```ts
export interface SprayGenerateMasksRequest {
  config: AppConfig;
  plan: SprayPaintPlan;
  heightPreview: TerrainHeightPreview;
  crop: TerrainCropRegion;
}

export interface SprayGenerateMasksResponse {
  masks: SprayMaskMeshPayload[];
  warnings: string[];  // 如陡崖溢色
}

export interface SprayMaskMeshPayload extends TerrainMeshPayload {
  colorIndex: number;   // 1-based
  regionId: number;
  fileName: string;     // Mask_Color_01.stl
}
```

### 2. 高度场薄壳算法（PRD §7.3）

对每个 `regionId = k`：

1. **包络顶面 Z**：`Z_shell = Z_terrain + maskFitToleranceMm + maskShellThicknessMm`
2. **开窗**：`cellRegions[row,col] === k` 的格点，顶面 quad **不生成**
3. **侧壁**：外轮廓 + 开窗边界，从 `Z_terrain + maskFitToleranceMm` 拉到 `Z_shell`
4. **底缘**：薄底环，`Z` 与 `Terrain_Main.bottomZ` 齐平或外扩 0.2 mm
5. **bleed**：`bleedMarginMm > 0` 时对 `regionId === k` 的 mask 做格点侵蚀/膨胀后再开窗

复用 [`shared/utils/heightfield-mesh.ts`](./shared/utils/heightfield-mesh.ts) 封边思路；**禁止** 对完整 `Terrain_Main` 做法向 offset + CSG。

### 3. 陡崖检测

- 坡度 > 70° 的格点若落在分区边界：收集 warning `"部分陡崖边界可能溢色，可调大 maskFitToleranceMm"`
- 通过 `SprayGenerateMasksResponse.warnings` 返回，任务-11 UI 展示

### 4. 主进程模块

```
electron/main/spray-paint/
  mask-shell-builder.ts   # 单 region 薄壳
  mask-generate-service.ts
```

IPC channel：`spray:generate-masks`（与 `spray:progress` 共用进度事件，`phase: "masks"`）。

### 5. 预览静态加载

`TerrainMeshPreview` 或子组件：

- 接收 `masks: SprayMaskMeshPayload[]`
- 默认 **不套合**，仅可在任务-11 前验证 mesh 可构建（可选：半透明叠加全部罩，或隐藏）
- 本任务最低要求：mesh 数据经 IPC 到达渲染进程并可 `BufferGeometry` 实例化，无破面 console 错误

### 6. 性能

- 4 个罩体串行或并行生成，主进程异步；典型 DEM 网格 < 30s
- 大网格可分批 progress 回调

## 验收标准

- [ ] 给定任务-09 的 plan，生成 4 个 `SprayMaskMeshPayload`
- [ ] 每个罩 `assertWatertightMesh` 通过
- [ ] 开窗格点与 `cellRegions === k` 一致（抽样校验）
- [ ] `maskFitToleranceMm` / `maskShellThicknessMm` 变更后几何随之变化
- [ ] `bleedMarginMm > 0` 时开窗边界相对收缩/扩张（P1）
- [ ] 陡崖场景返回 warnings
- [ ] 预览可加载 4 个罩体 mesh（不要求套合交互）

## 测试建议

1. 平坦地形 + 4 区：罩顶面平整、侧壁闭合
2. 圆盘裁剪：外轮廓侧壁闭合
3. 单区占 90%：其余开窗极少，罩几乎全封闭
4. `maskFitToleranceMm` 0.1 vs 0.5 对比内间隙
