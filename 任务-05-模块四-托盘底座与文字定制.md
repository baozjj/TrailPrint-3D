## 目标

生成包裹主模型的托盘式底座：可配置总厚度、下陷深度、边框宽度；在矩形/正多边形下支持分边纪念文字（阴刻/阳刻）。

## 功能清单

| ID | 需求项 | 说明 |
| --- | --- | --- |
| 4.1 | 托盘结构 | 总厚度、下陷深度、边框宽度 (mm)；校验：下陷深度 < 总厚度 |
| 4.2 | 边框文字 | 仅矩形/正多边形；每边独立：内容、居左/中/右、阴刻/阳刻、字体(3–5 种开源字体) |

## 验收标准

- [x] 4.1 三项参数可配置，非法组合（下陷≥总厚）表单校验失败并中文提示
- [x] 凹槽轮廓与任务-02 外形、尺寸一致，可容纳主模型外轮廓
- [x] 圆形底座不显示 4.2 文字配置
- [x] 矩形四边、正多边形 N 边各有独立文字配置组
- [x] 文字几何正确附着于边框（阴刻凹入 / 阳刻凸出）
- [x] 输出 `Tray_Base` 可打印封闭网格

## 技术要点

- 2D 文字轮廓 → 3D 挤出/布尔
- 开源字体授权清单与嵌入方式
- 与任务-02 形状参数的联动禁用逻辑

## 交付物

- `Tray_Base` 网格生成服务
- 参数：`totalThicknessMm`, `recessDepthMm`, `rimWidthMm`, `borderTextByEdge[]`
- 表单校验规则
