## 目标

生成可独立打印的轨迹实体模型，并在主模型表面按轨迹厚度执行挖槽；轨迹厚度即凹槽深度。

## 功能清单

| ID | 需求项 | 说明 |
| --- | --- | --- |
| 3.1 | GPX 轨迹优化 | 开关：过滤漂移点与毛刺 |
| 3.2 | 轨迹物理尺寸 | 宽度(mm)、厚度(mm)；厚度 = 主模型挖槽深度 |

## 验收标准

- [ ] 3.1 开启时轨迹点列经简化/滤波，关闭时保留原始解析结果
- [ ] 按 3.2 宽度挤出轨迹实体网格（`Trail_Line`）
- [ ] 主模型表面凹槽深度等于轨迹厚度
- [ ] 轨迹与地形贴合（沿 DEM 表面或投影规则文档化）
- [ ] 为任务-06 轨迹装配公差预留扩槽参数入口

## 核心逻辑

```
轨迹厚度(depthMm) === 主模型表面挖槽深度
轨迹实体用于分色打印；凹槽用于嵌入轨迹件
```

## 交付物

- `Trail_Line` 网格生成
- 主模型挖槽布尔运算
- 参数：`gpxSimplify`, `trailWidthMm`, `trailDepthMm`
