## 目标

根据裁剪区域与 DEM 生成带固定底厚、表面起伏的山体网格；支持 Z 轴夸张与平滑档位；输出可供挖槽与装配的主模型几何。

## 功能清单

| ID | 需求项 | 说明 |
| --- | --- | --- |
| 2.1 | 模型基础厚度 | mm，地形最低点到纯平底面的实体厚度 |
| 2.2 | Z 轴高程夸张 | 滑块 1.0x～5.0x |
| 2.3 | 地形平滑度 | 单选：原始 / 轻度 / 中度 / 高度 |

## 验收标准

- [x] 主进程根据任务-02 裁剪范围采样 DEM 并生成封闭网格
- [x] 2.1～2.3 参数变更可触发重新生成（带防抖或进度提示）
- [x] 模型底部为纯平面，厚度满足 2.1
- [x] 网格数据可通过 IPC 传回渲染进程做轻量 3D 预览
- [x] 为任务-04 预留轨迹槽挖除接口（表面布尔减）

## 技术要点

- DEM 数据源接入与重采样
- 平滑算法分档实现
- 主进程网格管线（禁止在渲染进程做重计算）

## 交付物

- `Terrain_Main` 网格生成服务
- 参数：`baseThicknessMm`, `zExaggeration`, `terrainSmoothing`
- 中间体：`Terrain_Main` 几何（挖槽前/后由任务-04、06 组合）
