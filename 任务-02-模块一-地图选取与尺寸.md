# 任务-02：模块一 — 地图选取与尺寸定义

| **状态**   | 待开始                                                |

## 目标

让用户选择底座外形与物理尺寸，并在带高程视觉的地图上通过遮罩平移/缩放，确定 3D 模型的 Bounding Box 与构图范围。

## 功能清单

| ID  | 需求项       | 说明                                                       |
| --- | ------------ | ---------------------------------------------------------- |
| 1.1 | 形状选择器   | 单选：圆形 / 矩形 / 正多边形                               |
| 1.2 | 物理尺寸     | 圆形→半径(mm)；矩形→长×宽(mm)；正多边形→边数(3–8)+边长(mm) |
| 1.3 | 交互地图视窗 | 底图含高程渲染；半透明 Mask；拖拽平移、滚轮缩放            |

## 验收标准

- [ ] 切换形状时，尺寸输入区按 1.2 联动显示/隐藏
- [ ] 遮罩外形与 1.1、1.2 一致，尺寸为底座最外围物理尺寸
- [ ] GPX 轨迹可通过平移/缩放调整至遮罩内理想位置
- [ ] 输出裁剪范围（中心、范围、形状参数）供主进程生成网格

## 技术要点

- 地图 SDK 或自定义 Canvas/WebGL 底图
- Mask 层与地图变换矩阵同步
- 将屏幕交互结果转换为地理/投影坐标下的裁剪多边形

## 交付物

- `MapCrop` 配置状态与 UI（可与任务-08 联调）
- 裁剪区域数据结构（供 DEM 采样与底座轮廓）
