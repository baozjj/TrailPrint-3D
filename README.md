# 🏔️ 迹印 TrailPrint 3D

**将你的徒步记忆，变成触手可及的 3D 打印地形纪念碑。**

「迹印 TrailPrint 3D」是一款专为户外爱好者与 3D 打印玩家打造的创新工具。只需上传你的 GPX 运动轨迹，即可在网页端轻松定制并导出可直接用于多色 3D 打印的实体地形模型。

告别单调的平面轨迹图，让每一次翻山越岭，都成为书桌上独一无二的专属艺术品。

---

## 🚀 快速开始

本项目为 **Electron + Vue 3** 桌面应用，使用 npm 管理依赖。

### 环境要求

- [Node.js](https://nodejs.org/) 18 或更高版本
- npm（随 Node.js 安装）

### 安装与启动

```bash
# 克隆仓库后进入项目目录
cd TrailPrint-3D

# 安装依赖
npm install

# 配置环境变量（见下方说明）
cp .env.example .env

# 启动开发模式（会打开 Electron 窗口）
npm run dev
```

### 环境变量

复制 `.env.example` 为 `.env`，并填入 [OpenTopography](https://portal.opentopography.org/requestService?service=api) API Key。地形高程数据依赖该服务：

```bash
OPENTOPOGRAPHY_API_KEY=你的密钥
VITE_OPENTOPOGRAPHY_API_KEY=你的密钥
```

可选：高精度制版或自定义分辨率较大时，可提高 V8 堆内存上限（单位 MB，默认 8192）：

```bash
TRAILPRINT_HEAP_MB=8192
```

### 其他命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 开发模式，热更新 |
| `npm run build` | 构建生产产物到 `out/` |
| `npm run preview` | 预览构建后的应用 |
| `npm run package` | 构建并打包为安装包（输出到 `release/`） |
| `npm run typecheck` | TypeScript 类型检查 |

## ✨ 核心功能亮点

### 🗺️ 1. 自由构图与地图裁剪

- **多形状选择：** 支持将地形裁剪为完美圆形、经典矩形或个性化正多边形。
- **精准比例尺：** 自由输入真实打印尺寸（如 150mm x 150mm），模型等比完美缩放。
- **交互式取景：** 在数字地图上自由平移、缩放，框选出轨迹最完美的视觉构图。

### ⛰️ 2. 沉浸式地形塑造

- **Z轴高程夸张：** 真实山体缩放后往往显得平缓。通过倍数滑块拉伸高度，重现“群山连绵”的视觉震撼。
- **地形表面平滑：** 自动处理地形毛刺与阶梯感，提供低、中、高不同档位的平滑选项，让打印出的山体表面更细腻。
- **轨迹优化过滤：** 一键过滤 GPX 数据中的 GPS 漂移和噪点，让路线线条更加顺滑自然。

### 🖼️ 3. 专属定制托盘底座

- **内嵌式托盘设计：** 自动生成包裹山体主模型的画框式底座，提升整体工艺品属性。
- **边缘纪念刻字：** 支持在底座四周添加专属文字（如：日期、公里数、山峰名称），并可选择“阴刻（凹进）”或“阳刻（凸出）”效果。

### 🧩 4. 硬核玩家专属：免胶水组装系统

专为 3D 打印工艺痛点设计的装配系统，分件打印更轻松：

- **精准打印公差：** 自由设定“轨迹槽”与“底座槽”的预留公差（如 0.15mm），确保耗材膨胀也能完美无缝拼接。
- **隐藏式磁吸设计：** 只需输入磁铁的直径与厚度，系统会自动在地形和底座内部生成精准对齐的孔位。无需一滴胶水，听到“啪”的一声即可完美吸附组装。
- **冰箱贴模式：** 可选择在底座最底部生成磁铁孔，让你的群山不仅能立于桌面，更能吸附于任何铁质表面。

## 🛠️ 操作流程 (How it Works)

1. **导入轨迹：** 上传由运动软件导出的 `.gpx` 格式文件。
2. **构图取景：** 选择底座形状，拖动地图使轨迹居中。
3. **细节定制：** 调整山体高度、刻上专属纪念文字。
4. **配置公差：** 根据你的 3D 打印机精度，设定拼装公差与磁吸孔位。
5. **一键导出：** 获取打包好的 3D 模型压缩包。

---

## 📦 交付物说明

点击生成后，您将获得包含以下 3 个高精度 `.stl` 文件的压缩包：

- **`Terrain_Main.stl`**：带有基础厚度、表面已精准挖去轨迹凹槽的山体主模型。
- **`Trail_Line.stl`**：应用了缩小公差的独立轨迹线条模型（完美适配换色打印）。
- **`Tray_Base.stl`**：带有内嵌凹槽、边缘刻字以及对齐孔位的托盘底座。

将它们分别导入切片软件，发挥你的色彩创意，开始打印吧！
