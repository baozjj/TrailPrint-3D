## 目标

实现 Apple HIG 风格的 Vue 3 界面：沉浸式预览区 + 左侧磨砂配置栏，简体中文文案，参数与各功能模块状态双向绑定。

## 布局结构

| 区域 | 要求 |
| --- | --- |
| 预览区（右/背景） | 全高地图/3D 占位；顶部「2D 地图视图 \| 3D 模型预览」分段控件 |
| 配置栏（左浮动） | 玻璃态、可滚动；标题 + 导入 GPX；五段手风琴；底部固定「生成并下载 STL」 |

## 配置模块（与 PRD 对齐）

1. **地图与尺寸** — 形状分段控件、动态尺寸输入 (mm)
2. **地形塑造** — 基础厚度、Z 夸张滑块 1.0x–5.0x、平滑度分段控件
3. **轨迹设置** — GPX 优化开关、宽度/深度并排输入
4. **托盘底座** — 总厚/下陷/边框
5. **装配与磁铁** — 默认折叠；公差输入；磁铁开关与孔位复选框

## 设计系统要点

- 字体：Inter / SF；次要文字 `text-gray-500`
- 组件：iOS 分段控件、Toggle、细滑块、圆角输入、胶囊主按钮（黑底）
- 语言：**全部 UI 文案为简体中文**
