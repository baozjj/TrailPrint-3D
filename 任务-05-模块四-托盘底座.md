## 目标

生成包裹主模型的托盘式底座：可配置总厚度、下陷深度、边框宽度。

## 功能清单

| ID | 需求项 | 说明 |
| --- | --- | --- |
| 4.1 | 托盘结构 | 总厚度、下陷深度、边框宽度 (mm)；校验：下陷深度 < 总厚度 |

## 验收标准

- [x] 4.1 三项参数可配置，非法组合（下陷≥总厚）表单校验失败并中文提示
- [x] 凹槽轮廓与任务-02 外形、尺寸一致，可容纳主模型外轮廓
- [x] 输出 `Tray_Base` 可打印封闭网格

## 技术要点

- 托盘外轮廓挤出 + 凹槽 CSG 差集
- 与任务-02 形状参数的联动

## 交付物

- `Tray_Base` 网格生成服务
- 参数：`totalThicknessMm`, `recessDepthMm`, `rimWidthMm`
- 表单校验规则
