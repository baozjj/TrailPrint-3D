## 目标

在 3D 预览弹窗内提供「喷漆分色」主入口：基于卫星图颜色 + 高程 + 坡度，离线将 DEM 格点映射为固定地表种类，生成 `cellRegions` 与色板，并在山体预览上即时分色显示。

## 固定种类（8 类）

| categoryId | 标签 | 说明 |
| --- | --- | --- |
| 0 | 山体岩石 | 裸露岩壁、碎石坡 |
| 1 | 建筑 | 屋顶、道路硬化面 |
| 2 | 水体 | 河流、湖泊 |
| 3 | 沙地 | 沙漠、沙滩 |
| 4 | 土路/裸地 | 土路、裸土、工地 |
| 5 | 雪地 | 积雪、冰川 |
| 6 | 植被 | 森林、草地 |
| 7 | 未知 | 未命中阈值兜底 |

### 首期 `colorCount=4` 与 8 类关系

分类层始终输出 `categoryId` (0–7)。喷漆导出层使用 `regionId` (0..`colorCount`-1)：

**默认合并表**（`categoryRuleVersion: 1`）：

| regionId | 合并的 categoryId | 默认标签 |
| --- | --- | --- |
| 0 | 6 | 植被 |
| 1 | 2, 5 | 水体/雪地 |
| 2 | 0, 3 | 山体/沙地 |
| 3 | 1, 4, 7 | 建筑/裸地/未知 |

`cellRegions` 存储 **regionId**（非 categoryId），供罩体与预览直接使用。

## 技术要点

### 1. 主进程卫星图采样

渲染进程已有 [`src/utils/satellite-imagery.ts`](./src/utils/satellite-imagery.ts)（ESRI 瓦片拼接）。主进程需等价实现或抽取共享模块：

- 输入：`mapCrop`、`viewport`、`gridCols/Rows`、裁剪区 `TerrainCropRegion`
- 输出：与格点对齐的 `RGB` 数组（行优先，长度 `cols × rows`）
- 瓦片拉取失败时：降级为仅高程+坡度规则分色，UI 提示「卫星图不可用，已使用地形规则」

### 2. 映射算法（PRD §7.2）

对每个格点 `(row, col)`：

1. 特征：`[R, G, B, elevationNorm, slopeNorm]`
2. RGB → Lab，与各类别基准色算距离；结合高程/坡度加权打分
3. 取得分最高 `categoryId`；低于阈值 → categoryId 7
4. 冲突优先级：水体 > 雪地 > 建筑 > 沙地 > 土路/裸地 > 山体岩石 > 植被 > 未知
5. 经默认合并表转为 `regionId`
6. 后处理：3×3 多数投票平滑；面积 < 0.5% 孤岛合并到邻域众数

**类别基准色（可配置常量，v1 默认值）**：

```ts
const CATEGORY_ANCHORS_LAB = {
  rock:    { L: 45, a: 2,  b: 8  },
  building:{ L: 55, a: 0,  b: -2 },
  water:   { L: 35, a: -5, b: -25},
  sand:    { L: 75, a: 2,  b: 25 },
  dirt:    { L: 50, a: 5,  b: 15 },
  snow:    { L: 92, a: -1, b: 2  },
  vegetation:{ L: 42, a: -12, b: 10 },
};
```

高程/坡度辅助规则示例：

- `elevationNorm > 0.75 && slopeNorm > 0.5` → 山体岩石加分
- `elevationNorm < 0.15 && slopeNorm < 0.2` → 水体加分
- `elevationNorm > 0.6 && waterScore 低` → 雪地候选（结合 RGB 高亮）

### 3. 预览着色（PRD §7.4）

`cellRegions` 在规则格点上；`buildHeightfieldTerrainMesh` 顶点可能不一一对应：

- 顶面顶点按 `(x,y)` 反查最近格点取 `regionId`，或
- 格点色纹理 / 三角重心采样

推荐：在 `TerrainMeshPreview` 为顶面 mesh 写 vertex colors 或 `CanvasTexture` 采样 `cellRegions`。

### 4. 类型定义

新建 [`shared/types/spray-paint.ts`](./shared/types/spray-paint.ts)：

```ts
export type SprayPaintSource = "rules" | "manual";

export interface SprayColorSlot {
  index: number;       // 1-based → Mask_Color_01
  hex: string;
  label: string;
  description?: string;
  regionId: number;
}

export interface SprayPaintPlan {
  colors: SprayColorSlot[];
  cellRegions: Uint8Array;
  gridCols: number;
  gridRows: number;
  source: SprayPaintSource;
  generatedAt: number;
}

export interface SpraySegmentRequest {
  config: AppConfig;
  heightPreview: TerrainHeightPreview;
  crop: TerrainCropRegion;
  viewportWidth: number;
  viewportHeight: number;
}

export interface SpraySegmentProgress {
  phase: "satellite" | "segment" | "smooth" | "done";
  progress: number;
  message: string;
}
```
