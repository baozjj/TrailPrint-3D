## 目标

在 3D 预览中提供遮挡罩 **套合检查** 交互：列出 N 个罩体，点击切换套合显示，支持视图模式与色罩联动，帮助用户确认喷涂开窗区域正确。

## 范围

### 包含

- 罩体列表 UI（名称、套合开关）
- 点击罩体 → 套合显示（半透明对齐山体）
- 视图模式：山体分色 / 山体+当前罩 / 仅当前罩
- 选中罩与色板高亮联动
- 显示任务-10 返回的 warnings（陡崖溢色等）
- 隐藏/显示全部罩体（P1）

### 不包含

- 重新分色算法（任务-09）
- 罩体 mesh 生成（任务-10）
- ZIP 导出（任务-12）

## 技术要点

### 1. 状态模型（渲染进程）

```ts
type SprayMaskViewMode = "terrain-colors" | "terrain-plus-mask" | "mask-only";

interface SprayMaskUiState {
  viewMode: SprayMaskViewMode;
  activeMaskIndex: number | null;  // 1-based colorIndex，null = 无选中
  showAllMasks: boolean;
  fitMaskIndex: number | null;     // 当前套合的罩
}
```

建议放在 `TerrainPreviewModal` 或专用 composable `useSprayMaskPreview.ts`。

### 2. 套合显示

- 套合时罩体 mesh 与山体 **同变换**（原点中心、Z 向上），无额外 offset
- 材质：半透明 `MeshStandardMaterial`，`opacity ~0.45`，`depthWrite: false` 可选以减少 z-fighting
- 非套合罩体：隐藏或极低透明度

### 3. 视图模式

| 模式 | 山体 | 当前罩 | 其他罩 |
| --- | --- | --- | --- |
| 山体分色 | 分色可见 | 隐藏 | 隐藏 |
| 山体+当前罩 | 分色可见 | 半透明套合 | 隐藏 |
| 仅当前罩 | 隐藏或线框 | 不透明/半透明 | 隐藏 |

切换用 iOS 风格分段控件，文案简体中文。

### 4. 色罩联动

- 色板行点击 ↔ 罩体列表行点击：同步 `activeMaskIndex`
- 高亮：边框/背景 accent + 3D 中对应 region 顶面闪烁或加亮（可选简单 outline）

### 5. 组件改动

| 文件 | 改动 |
| --- | --- |
| `TerrainPreviewModal.vue` | 罩体列表、视图分段、warnings 条 |
| `TerrainMeshPreview.vue` | 接收 `masks`、`sprayUiState`；按模式渲染 |
| 新建 `SprayMaskList.vue`（可选） | 列表 + 套合 toggle |

### 6. 流程串联

用户路径：

1. 任务-09「规则分色」→ 山体 4 色
2. 自动或手动触发「生成遮挡罩」（调用任务-10 IPC）
3. 罩体列表出现 4 项
4. 点击 `Mask_Color_02` → 套合预览，确认开窗区域

「生成遮挡罩」按钮可放在喷漆面板，本任务实现 UI 与交互；生成逻辑调用任务-10。
