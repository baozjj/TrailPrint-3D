# 边框刻字字体（可选）

将开源字体文件放入此目录后，任务-07 STL 导出可接入 `opentype.js` 矢量轮廓（当前预览使用内置笔画字模）。

| 文件 | 字体 | 许可 |
| --- | --- | --- |
| `Inter-Regular.ttf` | Inter | OFL-1.1 |
| `RobotoSlab-Regular.ttf` | Roboto Slab | Apache-2.0 |
| `NotoSansSC-Regular.otf` | Noto Sans SC | OFL-1.1 |
| `SourceSans3-Regular.otf` | Source Sans 3 | OFL-1.1 |
| `Literata-Regular.ttf` | Literata | OFL-1.1 |

`fontId` 与文件名对应关系见 `shared/tray/font-catalog.ts`。
