## 目标

通过公差与可选磁铁孔位，使三件分色打印部件可免胶水拼装且定位可靠。

## 功能清单

| ID | 需求项 | 说明 |
| --- | --- | --- |
| 5.1 | 3D 打印公差 | 轨迹槽公差（默认 0.15mm）：凹槽比轨迹实体大；托盘槽公差（默认 0.20mm）：底座凹槽比主模型外轮廓大 |
| 5.2 | 磁铁孔位 | 开关；直径/厚度(mm)；孔位一：底座底面浅孔（冰箱贴）；孔位二：底座凹槽底面 + 主模型底面配对孔（免胶水拼接） |

## 验收标准

- [x] 5.1 公差应用于对应布尔偏移，默认值可改
- [x] 5.2 关闭时不生成任何磁铁孔
- [x] 孔位一仅在勾选时出现在 `Tray_Base` 底面
- [x] 孔位二勾选时，底座凹槽底与主模型底面孔位坐标一一对应
- [x] 三件模型合并预览或装配说明可验证间隙设计

## 核心逻辑

```
轨迹凹槽宽度/深度补偿 += trailFitToleranceMm
托盘凹槽相对主模型外轮廓 += trayFitToleranceMm
磁铁孔：圆柱布尔减，深度与直径由用户规格决定
```

## 交付物

- 公差与磁铁参数写入网格管线
- 更新后的 `Terrain_Main`、`Trail_Line`、`Tray_Base` 几何
